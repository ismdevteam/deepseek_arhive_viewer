📖 How to Use This Viewer

Step‑by‑step guide

  1. Export your conversations from chat.deepseek.com
    • Go to chat.deepseek.com and log in.
    • Click on your profile avatar (top right) → Settings.
    • Under Data or Privacy, find Export data (or Download conversations).
    • Request the export – you'll receive a conversations.json file (may be large, up to 100+ MB).
  2. Load the file into the viewer
    • Click the 📂 Load conversations.json button in the top bar.
    • Select your downloaded conversations.json file.
    • The viewer will parse it (takes a few seconds for large files) and show all conversations.
  3. Navigate and search
    • Conversation list – click any conversation to view messages.
    • Global search – search across all conversation titles and content (top bar).
    • In‑chat search – type in the “🔎 Case‑sensitive (min 3)” box to highlight text inside the current conversation.
    • Message navigation – use the Top / Prev / Next / Bottom buttons or arrow keys (↑ ↓).
    • Copy message – click “📋 Copy Msg” to copy the currently visible message. “📚 Copy All” copies the whole conversation.
    • Bookmarkable URLs – the URL updates with ?conversation=5&message=12 so you can share or reload exact positions.
💡 Tip: The viewer automatically remembers your last loaded file (via IndexedDB). After you load a file once, it will reload on next visit – no need to select it again.
🔧 Scrolling offset – If the message labels are hidden behind the top bar, you can adjust SCROLL_OFFSET_PX at the top of the HTML file (currently 128px).

✅ That's it! Enjoy exploring your DeepSeek chat history.
